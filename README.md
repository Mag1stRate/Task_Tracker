# Task Tracker - 命令行任务管理工具

一个用Go语言实现的命令行任务管理工具，支持添加、修改、删除和查看任务，数据持久化存储在JSON文件中。

---

## 📋 项目介绍
这是一个基于命令行的任务追踪应用，用户可以通过以下方式管理任务：
- 添加任务（状态：待办/进行中/已完成）
- 修改任务状态
- 删除任务
- 查看所有任务
数据保存在本地的`tasks.json`文件中。
